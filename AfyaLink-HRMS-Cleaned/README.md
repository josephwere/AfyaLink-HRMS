# AfyaLink-HRMS - Cleaned

## Setup

Backend:
1. cd backend
2. cp .env.example .env and fill values
3. npm install
4. npm run dev

Frontend:
1. cd frontend
2. cp .env.example .env
3. npm install
4. npm start

This cleaned project standardizes backend to CommonJS, adds role-based middleware, getIO helper for Socket.io, and simplified routes structure.
